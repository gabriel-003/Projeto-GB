## Controles

| Tecla | Ação |
|---|---|
| `W` / `↑` | Mover Norte |
| `S` / `↓` | Mover Sul |
| `A` / `←` | Mover Oeste |
| `D` / `→` | Mover Leste |
| `Q` | Diagonal Noroeste |
| `E` | Diagonal Nordeste |
| `Z` | Diagonal Sudoeste |
| `C` | Diagonal Sudeste |
| `R` | Reiniciar o jogo |
| `ESC` | Sair |

## Objetivo
Coletar todas as moedas.
